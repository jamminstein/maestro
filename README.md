# maestro

> Orchestral generative system for OP-1/OP-Z/OP-XY

## Controls

- **E1** — tempo
- **E2** — mode (Lydian/Dorian/Mixolydian/Aeolian/Phrygian)
- **E3** — density
- **K2** — play / stop
- **K3** — new section / advance chord progression
- **K1 (hold)** — maestro page (mode select, root, density, dynamics, gestures)

## Grid

8 voices with mute/activity per row; maestro page for orchestration parameters, root selection, and gesture controls (swell, release, hush, cadence).

## Requirements

- norns
- grid (16×8)
- MIDI devices: OP-1 (port 1), OP-Z (port 2), OP-XY (port 3)
- engine: MollyThePoly (internal voices)
